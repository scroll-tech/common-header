## Available Scripts

### Build the library

```
yarn run rollup
```

### Publish the library

```
yarn publish
```

### Run tests

```
yarn run test
```

### Run storybook locally

```
yarn run storybook
```

### Build storybook

```
yarn run build-storybook
```
